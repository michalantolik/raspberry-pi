# Instalacja systemu, RPi jako PC

**NOOBS** -  **N**ew **O**ut **O**f the **B**ox **S**oftware

**Raspbian** - nazwa starszych wersji OS dla Raspberry Pi

**Raspberry Pi OS** - nazwa nowszych wersji OS dla Raspberry Pi

---

## Z użyciem monitora i instalatora NOOBS

### **Instalacja NOOBS**
1. Ściągnij NOOBS Lite - plik ZIP
- *https://www.raspberrypi.org/downloads/noobs/*
2. Rozpakuj ZIP i przenieś pliki na sformatowaną kartę SD

### **Instalacja Raspberry Pi OS**
3. Przełóż kartę SD z NOOBS Lite do Raspberry Pi
4. Podłącz monitor, mysz i klawiaturę do Raspberry Pi
5. Uruchom Raspberry Pi *(podłącz zasilacz)*
6. Na ekranie pojawi się instalator NOOBS Lite
7. Połącz się z wybraną siecią Wi-Fi
8. Zainstaluj Raspberry Pi OS

---

## Z użyciem monitora, ale bez użycia instalatora NOOBS *(z obrazu systemu)*
1. Ściągnij obraz wybranego systemu Raspberry Pi OS
- *https://www.raspberrypi.org/software/*
2. Ściągnij Etcher lub Rufus do nagrania obrazu Raspberry Pi OS na kartę SD:
- *https://www.balena.io/etcher/*
- *https://rufus.ie/*
3. Nagraj obraz Raspberry Pi OS na kartę SD
6. Przełóż kartę do Raspberry Pi
7. Uruchom Raspberry Pi *(podłącz zasilacz)*
8. Zainstaluj Raspberry Pi OS
